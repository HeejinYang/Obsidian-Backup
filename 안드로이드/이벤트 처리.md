발생한 이벤트를 버튼과 같은 객체에 전달한후 객체에게 처리과정을 위임한다

onClick()은 눌렀다가 뗄때 이벤트 발생
onTouch()는 motionEvent 매개변수(???)를 이용해서 더 자세하게 이벤트를 줄 수 있다

res/layout-land 폴더에 xml넣어서 가로방향일때 화면 생성 가능


- 버튼을 눌렀을때 함수를 호출하는 두가지 방식
1.버튼 id를 찾은후 (R.id.findViewById) 클릭리스너를 등록해서 클릭리스너에서 실행할 함수를 정의한다
2.xml파일에서 andriod:onClick속성에 직접 호출할 함수이름을 지정하고 activity.java에서 호출할 함수를 정의한다 / 간단한 클릭 이외에는 잘 안쓰임
