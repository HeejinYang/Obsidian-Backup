## 배열

데이터를 나열, 정리하기 위한 제일 기본적인 데이터구조, 자료구조

```javascript
const week = ["mon", "tue", "wed", "thu", "fri", "sat"];

console.log(week);
console.log(week[0]);

week.push("sun");
console.log(week);
week.pop();
console.log(week);

console.log(week[1341234]);
//이런식으로 로그 하면 undefined가 나온다

```
