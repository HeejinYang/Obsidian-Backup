브라우저의 '콘솔'로 자바스크립트 한줄씩 사용가능
js파일을 만들어서 긴 문장 실행시키자

```javascript
alert("hi");
```

브라우저가 자바스크립트 '실행'하게 해보기

그냥 js파일을 브라우저가 실행할수는 없다

브라우저는 HTML을 열고, <span style="color:red">HTML이 CSS와 자바스크립트를 가져온다</span>

```html
<body>
	<script src="app.js"></script>
</body>
```

> HTML은 접착제다!

> !+Tab -> HTML태그 자동완성
